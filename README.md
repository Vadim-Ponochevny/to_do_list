# ToDo List Application

Простое веб-приложение для управления задачами.

## Возможности

- Добавление новых задач
- Редактирование существующих задач  
- Удаление задач
- Поделиться задачами
- Автосохранение в локальное хранилище
- Защита от XSS-атак

## Технологии

- **HTML5** - структура приложения
- **CSS3** - стили и оформление
- **JavaScript (ES6+)** - логика и взаимодействие
- **Local Storage** - сохранение данных

## Установка и запуск

1. Скачайте файлы проекта
2. Откройте `index.html` в браузере
3. Начните добавлять задачи!

*Файл elements.html нужен был чтобы хранить все элементы для построения ui
